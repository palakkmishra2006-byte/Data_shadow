API Reference

Shadow Layer API

The Data-Shadow Proxy intercepts web queries and sanitizes outgoing telemetry payloads automatically. Developers can route application traffic directly through the local API proxy or integrate the client-side scripting to prevent tracker attribution.

Active Proxy Port

By default, the uvicorn server runs a middleware handler on the root port. Setting your client proxy to route requests through the core port enables header spoofing dynamically.

HTTP_PROXY=http://localhost:8000

Python requests Proxy Integration

Configure requests to mask client traffic. All outgoing traffic is routed through the proxy, which injects simulated cookie metrics.

python 
import requests

proxies = {
    'http': 'http://localhost:8000',
    'https': 'http://localhost:8000',
}

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64)',
    'Cookie': 'user_id=palak_admin_88; tracking_pixel_id=9928374'
}

# The Shadow layer will mask User-Agent and spoof target Cookie values
response = requests.get('https://analytics-collector.com/telemetry', 
                        headers=headers, 
                        proxies=proxies)

Node.js HTTP Agent Configuration

Force outgoing Node server request pipelines to utilize the Shadow Masking engine.

javascript 
const axios = require('axios');
const { HttpProxyAgent } = require('http-proxy-agent');

const proxyAgent = new HttpProxyAgent('http://localhost:8000');

axios.get('https://doubleclick.net/ads/pixel', {
    httpAgent: proxyAgent,
    headers: {
        'Cookie': 'uid=client_identity_original_token'
    }
}).then(res => {
    console.log("Telemetry sent safely (poisoned values injected).");
});
